# bgpSearcher
a BGP ASN and IPv4/IPv6 searcher for windows console
#
USAGE: Provide ASN or IPv4/Ipv6 Address to search\
  example 1: bgps 266725\
  example 2: bgps 200.26.106.0\
  example 3: bgps 2800:381::\
